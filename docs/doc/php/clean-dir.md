# clean\_dir

Cleans and normalizes a directory string.

```php
function clean_dir(string $dir): string
```

## Parameters

| Name | Type | Description |
| :--- | :--- | :--- |
| $dir | string | A directory. |

## Returns

| Type | Description |
| :--- | :--- |
| string | A cleaned directory. |

## Throws

| Type | Description |
| :--- | :--- |
| InvalidArgumentException | When the directory is invalid. |
