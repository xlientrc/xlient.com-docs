# markdown\_escape

Esacpes the specified string so that it won't interfere with markdown syntax.

```php
function markdown_escape(string $string): string
```

## Parameters

| Name | Type | Description |
| :--- | :--- | :--- |
| $string | string | A string to escape. |

## Returns

| Type | Description |
| :--- | :--- |
| string | An escaped string. |
